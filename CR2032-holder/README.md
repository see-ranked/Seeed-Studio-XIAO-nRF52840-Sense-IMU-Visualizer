- **電池タイプ**: CR2032ボタン電池
- **実装タイプ**: スルーホール
- **寸法**: 27.8mm × 22.8mm × 8.8mm (長さ × 幅 × 高さ)

## KiCadでの使用方法

### 1. シンボルライブラリの追加

1. KiCadのメインウィンドウで「設定」→「シンボルライブラリを管理」を開く
2. 「プロジェクト固有のライブラリ」タブを選択
3. 「追加」ボタンをクリック
4. 以下の情報を入力:
   - **ニックネーム**: `CR2032-Holder` (任意の名前)
   - **ライブラリパス**: `${KIPRJMOD}/CR2032-holder/CR2032-Holder.kicad_sym`
5. 「OK」をクリック

### 2. フットプリントライブラリの追加

1. KiCadのメインウィンドウで「設定」→「フットプリントライブラリを管理」を開く
2. 「プロジェクト固有のライブラリ」タブを選択
3. 「フォルダを追加」ボタンをクリック
4. `CR2032-holder`フォルダを選択
5. 以下の情報を確認:
   - **ニックネーム**: `CR2032-holder`
   - **ライブラリパス**: `${KIPRJMOD}/CR2032-holder`
6. 「OK」をクリック

### 3. 回路図での使用

1. 回路図エディタで「シンボルを追加」(Aキー)
2. フィルタに「CR2032」と入力
3. `CR2032_Holder`を選択して配置
4. リファレンス: `BT1`, `BT2`など

### 4. フットプリントの割り当て

1. 回路図エディタで「ツール」→「フットプリントを割り当て」
2. シンボルを選択し、フットプリントライブラリから`CR2032-Holder:CR2032_Holder`を選択
3. 「OK」をクリック

## ピン配置

| ピン番号 | 機能 | 説明 |
|---------|------|------|
| 1 | + | 正極 (左側のパッド) |
| 2 | - | 負極 (右側のパッド) |

## フットプリント仕様

- **パッドタイプ**: スルーホール、円形
- **パッドサイズ**: 直径2.0mm
- **ドリル径**: 1.2mm
- **パッド間隔**: 21.0mm (中心間)
- **シルクスクリーン**: 
  - 部品外形: 27.8mm × 22.8mm
  - 電池外形: 直径20mm (参考)
  - 極性マーク: "+" と "-"
- **コートヤード**: 部品外形 + 0.25mm

## 設計上の注意

1. **極性の確認**: 
   - パッド1(左側)が正極(+)
   - パッド2(右側)が負極(-)
   - シルクスクリーンに極性マークあり

2. **クリアランス**: 
   - 電池ホルダーの高さは8.8mmあるため、周辺部品との干渉に注意
   - コートヤードを確認して十分なスペースを確保

3. **実装**: 
   - スルーホール実装
   - 手はんだまたは波はんだ対応

4. **CR2032電池の仕様**:
   - 公称電圧: 3V
   - 直径: 20mm
   - 厚さ: 3.2mm
   - 容量: 約220mAh (メーカーにより異なる)

## トラブルシューティング

### シンボルが見つからない

- ライブラリパスが正しいか確認
- `${KIPRJMOD}`変数がプロジェクトフォルダを指しているか確認
- KiCadを再起動してライブラリを再読み込み

### フットプリントが見つからない

- フットプリントライブラリのパスが正しいか確認
- `.kicad_mod`ファイルが`CR2032-holder`フォルダ内にあるか確認

### 3Dビューで表示されない

- 3Dモデルは参考用の汎用モデルを指定しています
- 実際の形状と異なる場合がありますが、フットプリント自体は正確です
