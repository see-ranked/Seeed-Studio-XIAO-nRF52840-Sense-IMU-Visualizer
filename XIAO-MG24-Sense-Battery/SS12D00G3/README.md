# SS12D00G3 スライドスイッチ KiCadライブラリ

## 概要
秋月電子で購入可能なSS12D00G3小型スライドスイッチ用のKiCad v9.0ライブラリです。

## 部品仕様
- **型番**: SS12D00G3
- **タイプ**: SPDT (Single Pole Double Throw) スライドスイッチ
- **実装方法**: スルーホール
- **ピン間隔**: 2.5mm
- **本体サイズ**: 約8.6mm × 3.6mm × 3.0mm
- **購入先**: [秋月電子通商](https://akizukidenshi.com/)

## ファイル構成
```
SS12D00G3/
├── SS12D00G3.kicad_sym          # シンボルライブラリ
├── SS12D00G3.pretty/
│   └── SS12D00G3.kicad_mod      # フットプリント
└── SS12D00G3_README.md          # このファイル
```

## ピン配置
```
Pin 1 (NO)  ●─────┐
            │     │
Pin 2 (COM) ●─────┤ スイッチ本体
            │     │
Pin 3 (NC)  ●─────┘

ピン間隔: 2.5mm
```

### ピン機能
- **Pin 1**: NO (Normally Open) - 通常開
- **Pin 2**: COM (Common) - 共通端子
- **Pin 3**: NC (Normally Closed) - 通常閉

## KiCadでの使用方法

### 1. ライブラリの追加

#### シンボルライブラリ
1. KiCadを起動
2. **設定** → **シンボルライブラリを管理**
3. **プロジェクト固有のライブラリ**タブを選択
4. **フォルダアイコン**をクリック
5. `SS12D00G3.kicad_sym` を選択
6. ニックネームを `SS12D00G3` に設定

#### フットプリントライブラリ
1. **設定** → **フットプリントライブラリを管理**
2. **プロジェクト固有のライブラリ**タブを選択
3. **フォルダアイコン**をクリック
4. `SS12D00G3.pretty` フォルダを選択
5. ニックネームを `SS12D00G3` に設定

### 2. 回路図での使用
1. **シンボルを追加** (ショートカット: `A`)
2. フィルタに `SS12D00G3` と入力
3. シンボルを選択して配置

### 3. フットプリントの割り当て
1. シンボルを選択
2. **プロパティ** → **フットプリント**
3. `SS12D00G3:SS12D00G3` を選択

## 設計上の注意点

### 電気的仕様
- **定格電圧**: データシートを確認してください
- **定格電流**: データシートを確認してください
- **接触抵抗**: 通常50mΩ以下

### 基板設計
- **パッド径**: 1.5mm (ドリル径: 0.9mm)
- **クリアランス**: 周囲に最低2.5mm以上の空間を確保
- **操作スペース**: スイッチレバーの可動範囲を考慮

### 実装時の注意
1. **極性**: ピン1(NO)とピン3(NC)を間違えないように注意
2. **はんだ付け**: 過度な加熱を避ける(350℃、3秒以内推奨)
3. **機械的ストレス**: 基板実装後、スイッチに過度な力を加えない

## 使用例

### 電源切り替え
```
VCC ─────┬─── Pin 1 (NO)
         │
         └─── Pin 3 (NC) ─── GND
         
         Pin 2 (COM) ─── 負荷
```

### 信号切り替え
```
Signal A ─── Pin 1 (NO)
                 │
Signal B ─── Pin 3 (NC)
                 │
         Pin 2 (COM) ─── マイコン入力
```
