# XIAO MG24 Sense Battery Pogo Pin Contact

ポゴピン（スプリング式ピン）でXIAO MG24 Senseのバッテリーパッドに接触させるためのフットプリントです。

## 📋 概要

このフットプリントは、XIAO MG24 Senseの裏面にあるバッテリーパッド（BAT+、BAT-）にポゴピンで接触させるために設計されています。

### 用途
- **着脱可能な電源供給**: XIAO MG24 Senseをはんだ付けせずに電源供給
- **テスト治具**: プログラミングやテスト用の治具作成
- **充電ドック**: 充電ステーション/ドックの作成
- **開発用**: 繰り返し着脱が必要な開発環境

## 🔧 仕様

### パッド配置
- **パッド数**: 2個（BAT+、BAT-）
- **パッドタイプ**: 円形SMDパッド
- **パッドサイズ**: 直径2.0mm
- **パッド間隔**: 12mm（中心から±6mm）

### ボード寸法
- **幅**: 21.0mm（XIAO MG24 Senseと同じ）
- **高さ**: 7.0mm
- **コートヤード**: 22.0mm × 8.0mm

### 推奨ポゴピン仕様
- **ピン径**: 1.0mm～1.5mm
- **ストローク**: 1.0mm以上
- **接触力**: 50g～100g程度
- **例**: P75-E2シリーズ、P100-Eシリーズなど

## 📐 XIAO MG24 Senseバッテリーパッド位置

XIAO MG24 Senseの裏面（基板底面）には以下のパッドがあります：

```
        [XIAO MG24 Sense 裏面]
        
        ┌─────────────────────┐
        │                     │
        │   BAT+      BAT-    │  ← バッテリーパッド
        │    ●          ●     │
        │                     │
        └─────────────────────┘
        
        間隔: 約12mm
```

## 💡 使用方法

### 1. PCB設計

1. KiCadでフットプリントライブラリに追加
2. 回路図でバッテリー接続用のコネクタシンボルを配置
3. このフットプリント `XIAO-MG24-Sense-Battery-PogoPin` を割り当て
4. PCBレイアウトでXIAO MG24 Senseの真下に配置

### 2. ポゴピンの実装

#### 方法A: ポゴピンを基板に直接はんだ付け
1. ポゴピンをSMDパッドにはんだ付け
2. ピンの高さを調整（通常3～5mm）
3. XIAO MG24 Senseを上から押し当てる

#### 方法B: ポゴピンホルダーを使用
1. 3Dプリントでポゴピンホルダーを作成
2. ポゴピンをホルダーに挿入
3. ホルダーをPCBに固定

### 3. 接続確認

1. XIAO MG24 SenseをポゴピンPCBに配置
2. 軽く押し付ける（ポゴピンが圧縮される）
3. テスターで導通確認
4. 電源を供給してXIAOが起動することを確認

## ⚠️ 注意事項

### 電源仕様
- **バッテリー電圧**: 3.7V Li-Po推奨
- **充電電流**: 最大50mA（XIAO内蔵充電回路）
- **極性**: BAT+が正極、BAT-が負極（**逆接続厳禁**）

### ポゴピン選定
- **接触抵抗**: できるだけ低いもの（<100mΩ）
- **耐久性**: 10,000回以上の挿抜サイクル推奨
- **ストローク**: 十分な余裕を持たせる（1mm以上）

### 機械的設計
- **位置合わせ**: XIAO MG24 Senseの位置決めガイドを設ける
- **押し付け力**: 均等に力がかかるように設計
- **クリアランス**: 周辺部品との干渉に注意

## 🛠️ 設計例

### 充電ドックの作成

```
1. このフットプリントを使用してPCBを設計
2. USB Type-Cコネクタを追加（5V入力）
3. 3.7V→5V昇圧回路を追加（オプション）
4. ポゴピンをBAT+、BAT-パッドにはんだ付け
5. XIAO MG24 Senseを置くだけで充電開始
```

### テスト治具の作成

```
1. このフットプリントを使用
2. メインのXIAO-MG24-Senseフットプリントも配置
3. ポゴピンで電源供給
4. スルーホールでプログラミング/デバッグ
5. 着脱可能なテスト環境の完成
```
